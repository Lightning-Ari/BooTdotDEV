
# Todo List(TUIs)

This is Todo List TUIs (Terminal User Interfaces) application. This application is use 
to help you to orgaize your day or task. This application store data in Mark down format(.md).

### Interface

|---- Display _
               | 2 Section _____
				|------ 1. Right section (Display and writing)
				|------ 2. Bottom Section (Display Messages) & (Option/Funtion)

**Display:** This interface allways will be still and only the option of fuction will be changed.

**Top Section:** This will be 90% to 95% of the screen. Here all the do

**Bottom Section:** This will a small section which will cover only 5% to 10% of the screen. This will
only show the complition messages and error. Along with all the option.

**Funtion**
|---- Home _
|	    |-- New, Edit, Save, Delete, Exit
|---- New - Open a new blank page 
|---- Edit - If todo item selected then open the saved page
|---- Save - This funtion only work when a page is open.
|---- Delete - this funtion work when 
|					- todo item is open 
|					- todo item list was open
|					- This will delete the todo item.
|---- Exit - Exit the program in any situation after saving the todo item if any unsaved or still writing
|
|

**Top Section**
|---- View _
|	    |---- Display in List format of all the subject of todos
|	    |---- If no todo Created Then Display "Empty/ No Todo item is created/ Yet to create Todo item"
|	    |---- If todo item is there then using arrow to navigate, 
|		  Enter to open(continue editing) and Esc to back.
|	    |---- If any todo item open/view and any new created ask user to save before exit.
|	    |---- If the item is not save Display 
|
|---- Navigation _ 
|                 |------- Arrow Key (for direction Only Up and Down required)
|		  |------- Enter to confirm and open/View
|		  |------- Esc to back and cancel
